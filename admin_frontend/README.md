vue-admin-frontend
===============

> 基于vue-element-admin

## 主要新特性

* 基于easyswoole和vue-element-admin后台管理前后端分离方案
* 后台服务端使用easyswoole提供接口数据
* 后台前端页面使用vue-element-admin
* 简单使用权限验证（只验证前端页面/路由访问权限）

## 安装依赖
npm install

## 文档地址
[easyswoole](https://www.easyswoole.com/)
